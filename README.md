# mentat

> Mentat was a profession or discipline that was developed as a replacement to the computers and thinking machines

Mental math puzzles with auto-scaling difficulty. Made for fun for my own usage, still very much work in progress (but working!). [git.hanki.dev/mentat](https://git.hanki.dev/mentat)

## Features

- addition, subtraction, multiplication and division math puzzles
- auto-scaling difficulty with level system
- tracks high scores
- supports custom keybindings


## Technical

- stores data to local storage
- plain html/css/js, no libraries
